# ISO/IEC 27001 — Status

**Status:** Planned — not yet certified

Chameleon Eye does not currently hold ISO/IEC 27001 certification. ISO 27001 preparation is a planned milestone following commercial launch.

When this milestone is reached, the following information will be published here:

- Certified service scope
- Certificate number
- Certification body and accreditation body
- Issue date, expiry date, and surveillance cycle
- Availability of certificate summary

This document will be updated when certification is achieved. To discuss security requirements in the meantime, use the contact form or the Trust Pack request process at /trust/trust-pack.
